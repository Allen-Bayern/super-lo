# Super Lo

一个受`Python`内置函数启发的工具函数集合，旨在使`JavaScript/TypeScript`开发更加高效和 Pythonic。

## 特性

-   🚀 **类型安全**：完整的`TypeScript`支持，包含全面的类型定义
-   🛡️ **安全操作**：优雅处理边缘情况和`null / undefined`值
-   🔄 **集合支持**：支持各种`JavaScript`集合（对象、数组、`Map`、`Set`）
-   📚 **完整文档**：提供英文和中文详细文档

## 安装

```bash
npm install super-lo
# 或
yarn add super-lo
# 或
pnpm add super-lo
```
